# WPT-Coil-Simulation

供實驗室參考用的COMSOL模型跟MATLAB APP

**.rar檔案為Comsol模型** 

注意，因為上傳容量上限是25MB，為縮減檔案大小我已經把部分檔案的模擬結果刪掉了，需要結果的再自行按下計算就行了。

  1. N26130493_Report_WPT_Coil.rar 為螺旋繞線雙D模型
  2. 簡化模型_雙D.rar 為簡化模型雙D模型
  3. 簡化模型_圓.rar 為簡化模型圓型線圈
  4. 簡化模型_方.rar 為簡化模型方型線圈



**.txt檔案為參數資料**



**.mlapp是線圈座標計算機，推薦用**MATLAB 2025**以上版本執行**



**以上資料僅供參考**
